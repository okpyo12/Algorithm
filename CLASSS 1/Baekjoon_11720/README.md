### 문제

N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

---

### 입력

첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.

---

### 출력

입력으로 주어진 숫자 N개의 합을 출력한다.

---

### 예제 입력

| 예제 입력1 | 예제 출력1 |
| :--------: | :--------: |
| 1<br/>1 | 1 |

| 예제 입력2 | 예제 출력2 |
| :--------: | :--------: |
| 5<br/>54321 | 15 |

| 예제 입력3 | 예제 출력3 |
| :--------: | :--------: |
| 25<br/>7000000000000000000000000 | 7 |

| 예제 입력4 | 예제 출력4 |
| :--------: | :--------: |
| 14<br/>10987654321 | 46 |
