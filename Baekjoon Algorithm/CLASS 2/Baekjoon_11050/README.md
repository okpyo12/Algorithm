### 문제

자연수 (N)과 정수 (K)가 주어졌을 때 이항 계수 {N}{K}를 구하는 프로그램을 작성하시오.

---

### 입력

첫째 줄에 (N)과 (K)가 주어진다. (1 ≤ (N) ≤ 10, 0 ≤ (K) ≤ (N))

---

### 출력

 
{N}{K}를 출력한다. 

---

### 예제 입력

| 예제 입력1 | 예제 출력1 |
| :--------: | :--------: |
| 5 2 | 10 |
