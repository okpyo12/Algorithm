### 문제

정수가 담긴 배열 array와 정수 n이 매개변수로 주어질 때, array에 n이 몇 개 있는 지를 return 하도록 solution 함수를 완성해보세요.

---

### 입력

1 ≤ array의 길이 ≤ 100
0 ≤ array의 원소 ≤ 1,000
0 ≤ n ≤ 1,000

---

### 출력

array n result
[1, 1, 2, 3, 4, 5] 1 2
[0, 2, 3, 4] 1 0
